LCP Root Cause

Render Delay: 5,210 ms (92% of LCP) — Google Fonts (Open Sans) blocks text rendering. H1 heading exists in HTML from byte one but cannot paint until web font downloads from fonts.googleapis.com. No font-display: swap, no preconnect hints.

Fix: Self-host Open Sans as woff2 + add font-display:swap + preconnect. Expected LCP improvement: 3-4 seconds.
